Connect your Queue Campaign with your WordPress setup.

== Description ==

Use our Widget, Community, and all the Queue Technologies integrations within your WordPress setup.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/Queue-Linker` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->QueueTech page and set your Campaign Id.
4. now you can use our widget or the Shortcodes.

== Visual Editor ==
This plugin enables an extra button in the visual editor for use our SDK.

== Shortcodes ==

You can use several shortcodes to enable features within your posts or pages.

[queue-signup] 
[queue-social-proof]
[queue-greetings-bar]
[queue-community]

== Widget ==

The plugin add a new Widget in your Appearance->Widgets section.
